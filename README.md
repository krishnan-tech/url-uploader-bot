# url-uploader-bot

# Steps

- Clone Repo
- make .env file and write token like this

```
TOKEN=your_bot_token
SPOTIFY_CLIENT_ID=your_spotify_client_id
SPOTIFY_CLIENT_SECRET=your_spotify_client_secret
```

- run python main.py
